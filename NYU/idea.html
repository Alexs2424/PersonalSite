  • Sept 21

    • Gathering Recources Pt. 1

    This week I took some preliminary pictures with my smartphone of Subway signs, MTA cards,

    I was also paying attention closely to detail in the statiions because some of the signs are not

    Starting Images for Jay-St Metrotech Station

  • Sept 28

    • Gathering Recources Pt. 2

    I started cropping photos in a way that we could make 9/10 signs in every subway
    scannable. The photos need to be original and recognizeable. The MTA has similar graphic
    language throughout everything in the subway, so we can do a quick test on how the graphics can
    recognized and pull information. So I started working on a prototype using the MetroCard to see if
    the phone can scan a target and get information.

    Cropped Image for Recognition, So that All Signs Will Be Recognizeable

  • Oct 5

    • Proof Of Concept with Augmented Reality

    I wanted to start the development for the project, to make the common graphics used throughout the
    MTA's materials. I built a Unity project with Vuforia Cloud Recognition Services and trained it to recognize
    the common MTA card. This will be really easy to build off of for the subway signs inside. Shows that the technology
    is useful and totally doable.

    Cropped training AR recognition image

    This was cropped like this because it couldn't have anything behind in the scene. If you had some of the scene in
    the background then you could only find it once you were in that scene from that specific angle. This was the actual picture
    used to train the image recognition in the app. This is a proof of concept before I move to signs in the subway for next week.
